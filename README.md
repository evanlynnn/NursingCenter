## 运行项目

### 初始化数据库

创建名为nursing的数据库，并运行根目录下的nursing.sql

### 前端初始化运行

```
cd frontend
npm install
npm run serve
```

### 后端初始化

后端为Maven项目，使用IDEA打开，并运行build project.
